## 🎯 用户需求场景

### 场景 1：查看股票 K 线趋势
**用户行为：**
1. 打开网站 → 查看自选股列表
2. 点击某股票卡片的"📊 K 线"按钮
3. 查看日 K 线图（默认）

**期望结果：**
- K 线图立即显示（加载时间 < 2 秒）
- 显示最近 1 年数据
- 红涨绿跌，符合中国投资者习惯

### 场景 2：切换 K 线周期
**用户行为：**
1. 在 K 线弹窗中点击"周 K"按钮
2. 查看周线级别的趋势

**期望结果：**
- K 线图切换为周线
- 显示最近 3-5 年数据
- 均线、成交量同步更新

### 场景 3：查看技术指标
**用户行为：**
1. 勾选"MA5"、"MA10"、"MA20"
2. 查看均线与 K 线的关系

**期望结果：**
- 三条均线叠加在 K 线图上
- 颜色区分明显（MA5 橙色、MA10 蓝色、MA20 紫色）
- 可单独开关每条均线

### 场景 4：清理无效股票
**用户行为：**
1. 点击"🧹 清理无效股票"按钮
2. 查看确认框中的无效股票列表
3. 确认删除

**期望结果：**
- 显示无效股票的名称和代码
- 只删除真正无法获取数据的股票
- 不会误删正常显示的股票

### 场景 5：对比黄金价格
**用户行为：**
1. 查看网站黄金价格
2. 点击"📊 东方财富行情"按钮
3. 在新窗口查看东方财富黄金行情

**期望结果：**
- 新窗口打开东方财富 AU9999 页面
- 方便对比价格，验证准确性

## 🔧 技术架构

### 后端 API

| 接口 | 方法 | 参数 | 返回 | 用途 |
|------|------|------|------|------|
| `/api/kline` | GET | symbol, period, market | K 线数组 | K 线图数据 |
| `/api/stock/:symbol` | GET | symbol | 股票行情 | 单只股票详情 |
| `/api/stocks` | GET | symbols | 股票数组 | 批量获取行情 |
| `/api/stock/:symbol/info` | GET | symbol | 股票信息 | 验票功能 |
| `/api/search` | GET | keyword | 股票数组 | 股票搜索（保留） |
| `/api/market` | GET | - | 指数数组 | 大盘指数 |
| `/api/gold` | GET | - | 黄金价格 | 黄金监控 |

**数据源：**
- A 股/港股/美股：东方财富 API
- 大盘指数：腾讯财经 API
- 黄金价格：上海黄金交易所（通过华安黄金 ETF 518880）

### 前端组件

```
public/
├── index.html          # 主页面（K 线弹窗、搜票验票）

├── app.js              # 前端逻辑（自选股、预警、清理）
```

## ⚠️ 注意事项

### 1. 交易时间限制
- **A 股：** 周一至周五 9:30-11:30, 13:00-15:00
- **港股：** 周一至周五 9:30-12:00, 13:00-16:00
- **美股：** 周一至周四 21:30-次日 4:00，周五 21:30-次日 5:00（美东时间）
- **非交易时间：** 显示收盘价，不会实时更新

### 2. 数据源稳定性
- 东方财富 API 可能有请求频率限制
- 建议添加请求间隔（1 秒/次）
- 本地缓存 5 分钟，避免频繁请求

### 3. 网络要求
- 需要访问外部财经 API
- 如遇数据加载失败，检查网络连接
- GitHub 推送可能因网络不稳定失败（3 次失败即停止）

### 4. 数据延迟
- 行情数据可能有 1-3 秒延迟
- 仅供参考，不作为投资依据
