# 📈 股票盯票验票系统

一个基于 Web 的股票和黄金价格实时监控网站，支持自选股管理、价格预警和数据验证功能。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node](https://img.shields.io/badge/node-%3E%3D14.0.0-green.svg)
![Version](https://img.shields.io/badge/version-1.2.0-orange.svg)

## ✨ 功能特性

### 📊 大盘指数
- 实时监控上证指数、深证成指、沪深 300、创业板指
- 自动刷新，显示涨跌幅和成交量

### 🔍 股票搜索
- 支持股票代码和名称搜索
- 内置 20+ 热门股票（茅台、五粮液、平安、宁德时代等）
- 快速添加到自选列表

### 👁️ 自选股监控
- 添加/删除关注的股票
- 实时显示价格、涨跌幅、成交量、成交额
- 支持手动刷新和自动刷新（30 秒/次）
- 红涨绿跌，符合中国投资者习惯

### 🥇 黄金价格监控
- **实时金价** - 通过华安黄金 ETF (518880) 获取
- **数据源** - 紧密跟踪上海黄金交易所 AU9999 现货价格
- **自动更新** - 每 3 秒轮询一次
- **详细数据** - 开盘价、最高价、最低价、昨收价、涨跌幅

### ✅ 验票功能
- 验证股票代码有效性
- 显示股票基本信息（名称、代码、市场）
- 显示数据更新时间戳

### 🔔 价格预警
- 设置目标价格提醒
- 支持"高于"和"低于"两种条件
- 触发时高亮显示

### 📊 K 线图（v1.2.0 新增）
- 点击自选股卡片的"📊 K 线"按钮查看
- 支持周期切换：日 K、周 K、月 K、60 分钟 K
- 技术指标：MA5/10/20 均线、VOL 成交量
- 交互功能：缩放、拖拽、十字光标查看 OHLC
- 红涨绿跌，符合中国投资者习惯

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0

### 安装步骤

1. **进入项目目录**
```bash
cd D:\ClawXWorkDir\stock-monitor
```

2. **安装依赖**
```bash
npm install
```

3. **启动服务**

**方式一：使用 npm**
```bash
npm start
```

**方式二：直接运行 Node**
```bash
node server.js
```

**方式三：Windows 双击启动**
```bash
start-service.bat
```

4. **访问网站**
打开浏览器访问：http://localhost:3000

## 🔌 API 接口

### 获取单只股票行情
```
GET /api/stock/:symbol
```
**示例：** `/api/stock/600519`

**返回：**
```json
{
  "symbol": "600519",
  "name": "贵州茅台",
  "current": 1680.50,
  "open": 1675.00,
  "high": 1688.00,
  "low": 1672.00,
  "close": 1678.00,
  "change": 2.50,
  "changePercent": "0.15",
  "volume": 1234567,
  "amount": 2075000000,
  "time": "15:00:00",
  "date": "2026-03-26"
}
```

### 批量获取股票行情
```
GET /api/stocks?symbols=600519,000858,300750
```

### 搜索股票
```
GET /api/search?keyword=茅台
```

### 获取黄金价格
```
GET /api/gold
```

**返回：**
```json
{
  "symbol": "AU9999",
  "name": "黄金 9999",
  "current": 1017.70,
  "open": 1015.45,
  "high": 1022.50,
  "low": 1009.00,
  "close": 1015.45,
  "change": 2.25,
  "changePercent": "0.22",
  "time": "15:00:00",
  "market": "上海黄金交易所 (通过 ETF)",
  "etfPrice": 10.177
}
```

### 获取大盘指数
```
GET /api/market
```

## 💡 使用说明

### 添加自选股
1. 在搜索框输入股票代码或名称
2. 点击"搜索"按钮
3. 在搜索结果中点击股票，自动添加到自选列表

### 设置价格预警
1. 在"价格预警"区域输入股票代码
2. 输入目标价格
3. 选择"高于"或"低于"
4. 点击"设置预警"
5. 当价格触发条件时，预警项会高亮显示（红色背景）

### 验票功能
1. 在"验票"区域输入股票代码
2. 点击"验证"按钮
3. 查看股票详细信息和验证状态

### 自动刷新
- 点击"自动刷新：关"按钮开启
- 开启后每 30 秒自动刷新所有数据
- 再次点击关闭自动刷新

## 📊 数据源说明

| 数据类型 | 数据源 | 更新频率 |
|---------|--------|---------|
| A 股行情 | 新浪财经 API | 实时 |
| 大盘指数 | 腾讯财经 API | 实时 |
| 黄金价格 | 华安黄金 ETF (518880) | 3 秒/次 |
| 股票搜索 | 本地匹配 | - |

### 黄金价格计算
```
黄金现货价格 (元/克) = ETF 价格 (元/份) ÷ 0.01
```

华安黄金 ETF 每份约等于 0.01 克黄金，价格紧密跟踪上海黄金交易所 AU9999 现货价格。

## ⚠️ 注意事项

1. **交易时间**
   - 周一至周五：9:30-11:30, 13:00-15:00
   - 非交易时间显示收盘价，不会实时更新

2. **网络要求**
   - 需要访问外部财经 API
   - 如遇数据加载失败，请检查网络连接

3. **数据延迟**
   - 行情数据可能有 1-3 秒延迟
   - 仅供参考，不作为投资依据
