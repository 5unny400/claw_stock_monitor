**版本：** v1.2.0  
**制定日期：** 2026-03-26  
**优先级：** P0（核心功能）

---

## 📋 版本目标

在 v1.1.0（大盘监控、自选股、验票、黄金监控、价格预警）基础上，新增两大核心功能：

1. **📊 K 线图** - 支持日 K、周 K、月 K，技术指标（MA、VOL）
2. **📑 财报数据** - 显示核心财务指标（营收、净利润、PE、PB 等）

---

## 🎯 功能需求详述

### 一、K 线图功能

#### 1.1 功能描述
在自选股和验票页面，点击股票可查看 K 线图，支持：
- 切换周期：日 K、周 K、月 K、60 分钟 K
- 技术指标：MA（5/10/20 日）、VOL（成交量）
- 缩放、拖拽查看历史数据
- 最新价格实时高亮

#### 1.2 用户场景
| 场景 | 用户行为 | 期望结果 |
|------|---------|---------|
| 查看趋势 | 点击自选股卡片 → 查看 K 线 | 显示近 1 年日 K 线图 |
| 切换周期 | 点击"周 K"按钮 | K 线图切换为周线，显示近 3 年数据 |
| 看成交量 | 勾选"VOL"选项 | K 线图下方显示成交量柱状图 |
| 看均线 | 勾选"MA5/10/20" | K 线图叠加对应均线 |

#### 1.3 数据需求
**K 线数据 API（东方财富）：**
```
http://push2his.eastmoney.com/api/qt/stock/kline/get
参数：
- secid: 1.600519（沪市）/ 0.000858（深市）/ 116.00700（港股）
- klt: K 线类型（1=分钟，5=5 分钟，15=15 分钟，30=30 分钟，60=60 分钟，101=日 K，102=周 K，103=月 K）
- fqt: 复权类型（0=不复权，1=前复权，2=后复权）
- beg: 开始日期（20200101）
- end: 结束日期（20261231）
- fields1: f1,f2,f3,f4,f5,f6（K 线辅助字段）
- fields2: f51,f52,f53,f54,f55,f56,f57,f58,f59,f60（K 线核心字段）
```

**返回字段说明：**
```
f51: 日期（2026-03-26）
f52: 开盘价
f53: 收盘价
f54: 最高价
f55: 最低价
f56: 成交量
f57: 成交额
f58: 振幅
f59: 涨跌幅
f60: 涨跌额
```

#### 1.4 技术方案
**前端图表库：** Lightweight Charts（TradingView 开源，轻量、高性能）
- 体积：~50KB（gzip）
- 支持：K 线、成交量、指标叠加
- 交互：缩放、拖拽、十字光标

**替代方案：** ECharts（功能更全，但体积大 ~400KB）

**实现步骤：**
1. 安装依赖：`npm install lightweight-charts`
2. 新增 K 线 API：`GET /api/kline?symbol=600519&period=day`
3. 前端新增 K 线弹窗/页面，集成图表库
4. 添加周期切换、指标开关 UI

### 二、财报数据功能

#### 2.1 功能描述
在验票页面和 K 线图页面，可查看股票核心财务指标：
- 估值指标：PE（静/TTM）、PB、PS、市值
- 盈利能力：营收、净利润、毛利率、ROE
- 每股指标：每股收益（EPS）、每股净资产、每股现金流
- 分红信息：股息率、分红总额

#### 2.2 用户场景
| 场景 | 用户行为 | 期望结果 |
|------|---------|---------|
| 快速估值 | 验票后点击"查看财报" | 显示 PE、PB、市值等核心指标 |
| 对比分析 | 查看多只股票财报 | 横向对比营收、净利润增速 |
| 选股参考 | 设置筛选条件（PE<20, ROE>15%） | 筛选出符合条件的股票 |

#### 2.3 数据需求
**财报数据 API（东方财富）：**
```
http://push2.eastmoney.com/api/qt/stock/get
参数：
- secid: 1.600519
- fields: f9,f10,f11,f12,f13,f14,f15,f16,f17,f18,f19,f20...（财务字段）
```

**核心字段说明：**
```
f9: 市盈率（静）
f10: 市净率
f11: 市盈率（TTM）
f12: 股票代码
f13: 市值（元）
f14: 总市值
f15: 流通市值
f16: 总股本
f17: 流通股本
f18: 营收（元）
f19: 营收同比增长率
f20: 净利润（元）
f21: 净利润同比增长率
f22: 毛利率
f23: ROE（净资产收益率）
f24: 每股收益（EPS）
f25: 每股净资产
f26: 每股现金流
f27: 股息率
```

**季度财报 API（可选进阶）：**
```
http://datacenter.eastmoney.com/securities/api/data/get
参数：
- type: RPT_FCI_FINANCE
- sty: 财务指标字段
- filter: (SECUCODE="600519.SH")
```

#### 2.4 技术方案
**方案 A：实时 API（推荐 v1.2.0）**
- 优点：实现简单，数据实时
- 缺点：每次请求字段有限，需多次调用

**方案 B：缓存 + 定时刷新（v1.3.0 优化）**
- 每天收盘后（18:00）批量拉取财报数据
- 存储到本地 JSON 或 SQLite
- 前端读取缓存，减少 API 请求

**v1.2.0 采用方案 A**，原因：
- 开发成本低
- 数据实时性强
- 验证用户需求后再优化

## 🔧 技术架构

### 3.1 后端 API 新增接口

| 接口 | 方法 | 参数 | 返回 | 用途 |
|------|------|------|------|------|
| `/api/kline` | GET | symbol, period | K 线数组 | K 线图数据 |
| `/api/financials` | GET | symbol | 财报指标对象 | 财报概览 |
| `/api/financials/quarterly` | GET | symbol, year | 季度财报数组 | 历史财报（可选） |

### 3.2 前端新增组件

```
public/
├── kline-modal.html      # K 线图弹窗
├── kline-chart.js        # K 线图渲染逻辑
├── financials-panel.js   # 财报面板渲染逻辑
└── style.css（更新）     # 新增 K 线、财报样式
```

### 3.3 依赖安装

```bash
# K 线图表库（二选一）
npm install lightweight-charts    # 推荐，轻量
# 或
npm install echarts               # 功能全，体积大

# 可选：本地缓存（v1.3.0 用）
npm install better-sqlite3
```

## ⚠️ 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 东方财富 API 限流 | K 线数据加载失败 | 添加请求频率限制（1 秒/次），本地缓存 5 分钟 |
| K 线图表库兼容性问题 | 移动端显示异常 | 优先测试移动端，备选 ECharts |
| 财报字段缺失 | 部分指标无法显示 | 字段为空时显示"--"，不阻断页面 |
| 港股/美股 K 线数据格式不同 | 解析错误 | 分别测试 A 股/港股/美股，添加容错处理 |

## 📝 决策记录

### 已确定决策
1. **K 线图表库** → Lightweight Charts（轻量、专业）
2. **财报方案** → 方案 A（实时 API，v1.2.0 快速验证）
3. **UI 形式** → K 线弹窗 + 财报面板展开
4. **数据源** → 东方财富（统一 K 线、财报、行情）

### 待讨论决策
1. 是否需要本地缓存？（v1.2.0 先不做，根据性能决定）
2. 财报是否显示历史季度数据？（v1.2.0 只显示最新，v1.3.0 加历史）
3. K 线图是否支持画线工具？（v1.2.0 不支持，过于复杂）
