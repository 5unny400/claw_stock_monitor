# 📈 股票盯票验票系统 v1.3.0 产品方案

**版本：** v1.3.0（财报专题）  
**制定日期：** 2026-03-28  
**优先级：** P0（核心功能）

---

## 📋 版本目标

在 v1.2.0（K 线图、用户体验优化）基础上，新增**财报数据功能**，帮助用户快速了解股票基本面。

**核心目标：**
1. ✅ **财报数据展示** - PE、PB、营收、净利润、毛利率、ROE 等核心指标
2. ✅ **财报历史对比** - 近 4 季度营收/净利润趋势图
3. ✅ **选股器功能** - 按 PE、ROE、市值等条件筛选股票
4. ✅ **资金流向** - 主力资金、北向资金流入流出（可选）

**版本定位：** 从"技术面分析"扩展到"基本面分析"，提供完整的投资决策支持。

## 🎯 功能需求详述

### 一、财报数据功能（核心）

#### 1.1 功能描述
在验票页面和 K 线图页面，可查看股票核心财务指标：
- **估值指标：** PE（静/TTM）、PB、PS、市值
- **盈利能力：** 营收、净利润、毛利率、ROE
- **每股指标：** 每股收益（EPS）、每股净资产、每股现金流
- **分红信息：** 股息率、分红总额

#### 1.2 用户场景
| 场景 | 用户行为 | 期望结果 |
|------|---------|---------|
| **快速估值** | 验票后点击"查看财报" | 显示 PE、PB、市值等核心指标 |
| **对比分析** | 查看多只股票财报 | 横向对比营收、净利润增速 |
| **选股参考** | 设置筛选条件（PE<20, ROE>15%） | 筛选出符合条件的股票 |
| **趋势分析** | 查看近 4 季度财报趋势 | 营收/净利润增长曲线图 |

#### 1.3 数据需求
**财报数据 API（东方财富）：**
```
http://push2.eastmoney.com/api/qt/stock/get
参数：
- secid: 1.600519（沪市）/ 0.000858（深市）/ 116.00700（港股）
- fields: f9,f10,f11,f12,f13,f14,f15,f16,f17,f18,f19,f20...（财务字段）
```

**核心字段说明：**
```
f9: 市盈率（静）
f10: 市净率
f11: 市盈率（TTM）
f12: 股票代码
f13: 市值（元）
f14: 总市值
f15: 流通市值
f16: 总股本
f17: 流通股本
f18: 营收（元）
f19: 营收同比增长率
f20: 净利润（元）
f21: 净利润同比增长率
f22: 毛利率
f23: ROE（净资产收益率）
f24: 每股收益（EPS）
f25: 每股净资产
f26: 每股现金流
f27: 股息率
```

**季度财报 API（历史数据）：**
```
http://datacenter.eastmoney.com/securities/api/data/get
参数：
- type: RPT_FCI_FINANCE
- sty: 财务指标字段
- filter: (SECUCODE="600519.SH")
```

#### 1.4 技术方案
**方案：实时 API + 本地缓存**
- 优点：数据实时，减少 API 请求
- 实现：每天收盘后（18:00）批量拉取财报数据
- 存储：本地 JSON 文件缓存
- 前端：优先读取缓存，缓存过期则请求 API

**v1.3.0 采用方案**，原因：
- 平衡实时性和性能
- 减少 API 请求频率
- 支持历史数据对比

```
├── cashflow-panel.js       # 资金流向组件

```

### 数据缓存策略

**缓存文件：** `data/financials-cache.json`

**缓存逻辑：**
```javascript
// 每天 18:00 更新缓存
const CACHE_EXPIRY = 24 * 60 * 60 * 1000; // 24 小时

function getFinancials(symbol) {
  const cached = cache[symbol];
  if (cached && Date.now() - cached.timestamp < CACHE_EXPIRY) {
    return cached.data; // 返回缓存
  }
  
  // 缓存过期，请求 API
  const data = await fetchFromAPI(symbol);
  cache[symbol] = { data, timestamp: Date.now() };
  saveCache();
  return data;
}
```

## ⚠️ 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 东方财富 API 限流 | 财报数据加载失败 | 添加请求频率限制（1 秒/次），本地缓存 24 小时 |
| 财报字段缺失 | 部分指标无法显示 | 字段为空时显示"--"，不阻断页面 |
| 选股器性能问题 | 筛选大量股票时卡顿 | 前端分页，后端优化查询逻辑 |
| 数据准确性 | 财报数据与官方不一致 | 标注数据来源，误差>5% 时提示用户 |
